# 🚆 Online Reservation System

A Java Swing-based Online Reservation System developed as part of the **Oasis Infobyte Java Development Internship (Task 1)**.

The application allows users to log in, book train tickets, generate a unique PNR, fetch reservation details, and cancel reservations using the PNR number. The application uses **Java Swing** for the GUI, **JDBC** for database connectivity, and **MySQL** as the backend database.

---

## 📌 Features

- 🔐 User Login Authentication
- 🚆 Train Reservation Form
- 🚉 Auto-fetch Train Name using Train Number
- 🎫 Automatic PNR Generation
- 💾 Store Reservation Details in MySQL
- ✅ Booking Confirmation Dialog
- 🔍 Fetch Reservation using PNR
- ❌ Cancel Reservation
- ✔ Input Validation
- 🗃 MySQL Database Integration

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Import the MySQL Connector JAR.
4. Create the MySQL database and required tables.
5. Update database credentials in `DBConnection.java`.
6. Run `Main.java`.
